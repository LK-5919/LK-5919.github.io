平移-旋转-缩放

2022-03-12 HTML-CSS学习 681 字
WEB前端

元素的变形包括平移，旋转，缩放；

平移

通过开启 transform 属性中的 translateX(),translateY(),translateZ() 开启元素的平移效果；
	translateX() --- 开启元素的水平平移；
	translateY() --- 开启元素的垂直平移；
	translateZ() --- 开启元素的视距方向的平移（需要设置网页视距 perspective）
	
  .demo{
            width: 200px;
            height: 200px;
            background-color: #bfa;
            /* 开启元素的水平平移 */
            transform: translateX(50%) translateY(50%);
            /* 可知元素的平移是以自身的大小为标准计算，按照当前元素在文档流中的位置进行平移，而且平移的元素不脱离文档流
                也不影响其他元素 可以看成是提高了层级
                元素若还有其他的平移 直接写在同一个transform 里面 使用空格将每一个平移计算隔开
            */
        }

     width: 300px;
            height: 300px;
            background-color: #bfa;
            margin: 50px auto;
            /* 开启元素的Z轴平移 */
            transform: translateZ(200px);
            /* 在html设置网页视距perspective无效 
                需要在body标签设置perspective
                perspective一般在800px 到 1200px间
            */
        }

旋转

通过开启 transition 的rotateX() rotateY() rotateZ() 实现元素的 旋转效果；
	rotateX() --- 水平旋转 （想象成网页平面上的元素围绕着水平线旋转 是空间上的旋转 body标签开启perspective属性后 立体效果比较明显）
	rotateY() --- 垂直旋转 （想象成网页平面上的元素围绕着竖直线旋转）
	rotateZ() --- 围绕Z轴旋转（因为Z轴是指向我们的脸，所以此时网页平面上的元素旋转为在网页上的平面旋转 相当于俯视视角 此时无立体效果）

.demo{
            width: 320px;
            height: 320px;
            margin: 200px auto;
            transition:  all 5s;
        }
        .demo img{
            width: 320px;
            height: 320px;
        }
 body:hover .demo{

            /*
                通过旋转可以使元素沿着x y 或 z旋转指定的角度
                    rotateX()
                    rotateY()
                    rotateZ()
            */
            /* transform: rotateZ(.25turn); */
            /* transform: rotateY(180deg) translateZ(400px); */
            /* transform: translateZ(400px) rotateY(180deg) ; */
            transform: rotateX(1turn);
            /* 是否显示元素的背面 */
            /* backface-visibility: hidden; */
        }

缩放

通过开启 transition 中的 scaleX() scaleY() scale() 开启元素的缩放
	scaleX() --- 水平方向上的缩放；
	scaleY() --- 垂直方向上的缩放；
	scale() --- 双方向上的缩放；

 .demo{
            width: 200px;
            height: 200px;
            background-color: #bfa;
            display: inline-block;
            /* margin: 20px auto; */
        }
        .demo1{
            width: 200px;
            height: 200px;
            background-color: skyblue;
            display: inline-block;
        }
        .demo:hover{
            transform: scaleX(120%);
            /* 设置元素垂直方向上放大为自己的20% 
                可知此时元素的缩放不影响别的元素 也不脱离文档流
            	即此时的元素缩放不影响其他元素的所在文档流的位置
            */
        }

<div class="demo"></div>
<div class="demo1"></div>

 transform-style: preserve-3d;
            /* 只有开启transform-style 才能看出3D立体效果 */

注意：在测试时发现 translateZ() 也就是在视距方向上平移时 值为%表示不生效；
