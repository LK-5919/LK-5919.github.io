媒体查询

响应式布局

响应式布局 --- 网页根据浏览设备会窗口的大小呈现出不同的效果
		使用响应式布局是一个页面适用于所有设备
		通过媒体查询 -- 可以为不同的设备，或者设备的不同状态来分别设置样式

媒体查询

语法： @media 查询规则{}
		媒体类型：all -- 所有设备
				print -- 打印设备（只有页面在打印时样式才呈现出来）
				screen -- 带屏幕的设备
				speech -- 屏幕阅读器
		可以使用 , 连接多个媒体类型 表示一个 或 的关系
	也可以在媒体类型前添加一个 only 表示只有 （为了兼容一些老版本浏览器）

		媒体特性：width -- 视口宽度
				height -- 视口高度
				min-width -- 视口的最小宽度（当视口的宽度大于这个宽度时生效）
				max-width -- 视口的最大宽度（当视口的宽度小于这个宽度时生效）

		断点 --- 样式切换到额分界点 --- 网页样式发生变化的点
		常用断点：
				小于768 -- 超小屏幕 -- max-width = 768px
				大于768 -- 小屏幕 -- min-width = 768px
				大于992 -- 中型屏幕 -- min-width = 992px
				大于1200 -- 大屏幕 -- min-width = 1200px

<style>
 @media only screen and (min-width: 500px) and (max-width:700px){
     /*
     	当视口的宽度大于500px且视口的宽度小于700px时 body的样式如下：
     */
             body{
                background-color: #bfa;
             }
         }
  </style>
