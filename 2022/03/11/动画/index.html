过渡与动画

2022-03-11 HTML-CSS学习 1.2k 字
WEB前端

过渡

通过过渡指定一个属性发生变化时的切换方式；

transition-property : 属性指定要执行的过渡属性（大部分属性都支持过渡效果，注意过渡时必须是从一个有效数值向另外一个有效数值进行过渡，若属性值是 auto 则过渡无法生效）;
	transition-property : height,width,all; 多个属性值使用逗号隔开，all 关键字则指定所有可计算的属性；

transition-duration ： 指定过渡效果的持续时间 单位 ： s 和 ms ；

transition-timing-function ：过渡的时序函数 --- 指定过渡的执行速度方式；
	可选值：
		ease 默认值 --- 慢速开始 先加速后再减速
		linear 匀速运动
		ease-in 加速运动
		ease-out 减速运动
		ease-in-out 先加速后减速
		cubic-bezier() 来指定时序函数  
		steps() 分布执行过渡效果 --- 可以设置两个值（步数，end/start）
			end --- 在时间结束时执行过渡（默认值）
			start --- 在时间开始时执行过渡
	即使指定不一样的过渡时序函数执行时间都是按transition-duration指定

transition-delay ： 过渡效果的延迟，等待一段时间后再执行过渡，单位 ： s ms 


 .demo{
           width: 700px;
           height: 700px;
           background-color: grey;
           overflow: hidden;
        }
        .demo1{
            width: 100px;
            height: 100px;
            background-color: #bfa;
            /* 设定demo1开始的位置 */
            margin-left: 0;
            /* 设置demo1过渡的属性 --- margin-left */
            transition-property: margin-left;
            /* 设置demo1的过渡持续效果 */
            transition-duration: 2s;
            /* 设置demo1的过渡时序函数 --- 默认值*/
            transition-timing-function: ease;
        }
        .demo:hover div{
            margin-left: 600px;
        }
        .demo2{
            width: 100px;
            height: 100px;
            margin-top: 200px;
            background-color: #bfa;
            /* 设定demo1开始的位置 */
            margin-left: 0;
            /* 设置demo1过渡的属性 --- margin-left */
            transition-property: margin-left;
            /* 设置demo1的过渡持续效果 */
            transition-duration: 2s;
            /* 设置demo1的过渡时序函数 --- 使用曲线定义*/
            transition-timing-function: cubic-bezier(0,1.65,1,-0.77);
            /* 设置demo2延迟两秒再开始过渡 */
            transition-delay: 2s;
        }
        .demo3{
            width: 100px;
            height: 100px;
            margin-top: 200px;
            background-color: #bfa;
            /* 设定demo1开始的位置 */
            margin-left: 0;
            /* 设置demo1过渡的属性 --- margin-left */
            transition-property: margin-left;
            /* 设置demo1的过渡持续效果 */
            transition-duration: 2s;
            /* 设置demo1的过渡时序函数 --- steps*/
            transition-timing-function: steps(3,end);
            /* 指定过渡的时序为步骤的话 过渡效果会将整个默认过程均分为几个步骤 只展现那几步的状态
            */
        }

transition 属性可以同时设置过渡的相关属性，只要求持续时间要写在延迟时间之前，其他属性顺序随便；

动画

动画：动画和过渡类似，都可实现一些动态的效果；
	两者不同的是过渡需要在某个属性发生变化时才触发执行，而动画可以自动触发执行；

animation-name : 要对当前元素生效的关键帧名字；
animation-duration ：动画执行的时间；
animation-delay ： 动画执行的延迟时间；
animation-timing-function ：动画执行的时序函数；
animation-iteration-count ：动画的执行次数；
	可选值：次数（具体数值）；  infinite（无限执行）；
animation-direction ：指定动画的执行方向；
	可选值：normal --- 默认值 从from 到 to 关键帧执行 每次都是这样；
			reversal --- 从 to 到 from 关键帧执行 每次都是这样；
			alternate --- 从 from 到 to 执行关键帧 重复执行时反向执行上一次的动画；
			alternate-reverse --- 从 to 到 from 运行 重复执行动画时反向执行；
animation-play-state ： 设置动画的执行状态；
	可选值： running --- 默认值 动画执行； paused --- 动画暂停；
animation-fill-mode ： 动画的填充模式；
	可选值：none --- 默认值 动画执行完毕后元素回到原位置；
			forwards --- 动画执行完毕元素停在结束的位置；
			backwards --- 动画延时等待时，元素就会处于开始位置；
			both --- forwards 和 backwards 结合

.demo{
            width: 700px;
            height: 700px;
            background-color: #bfa;
        }
         /* 设置关键帧 */
         @keyframes demo1-move{
                /* from表示动画开始的位置和状态 */
                from{
                    margin-left: 0;
                    background-color: aqua;
                }
                /*  to 表示动画结束的位置 也可以用100% 表示 */
                to{
                    background-color: orange;
                    margin-left: 600px;
                }
            }
        .demo1{
            width: 100px;
            height: 100px;
            background-color: skyblue;
            /* 使用刚设置的关键帧 */
            animation-name: demo1-move;
            /* 设置动画的执行时长 */
            animation-duration: 3s;
            /* 设置动画的执行次数 迭代次数*/
            animation-iteration-count: 3;
            /* 设置动画的执行时序函数 */
            animation-timing-function: ease;
            /* 设置动画的执行方向 交替反向执行*/
            animation-direction: alternate;
            /* 设置动画的启动延时 */
            animation-delay: 1s;
            /* 设置动画的执行状态 */
            /* animation-play-state: paused; */
            /* 设置动画的执行填充  执行完以后停在执行结束的位置 */
            animation-fill-mode: both;
        }

animation 属性：简写形式 除了执行时长必须要写在延时启动时间之前 其他属性设置顺序随意；
