009-字体和图标

1.默认字体格式

设置网页字体属性 
	1. color 设置元素的字体颜色
	2. font-size 设置字体的大小 单位 px em（以父元素为标准） rem（以根元素body为标准）
		1em / 1rem =1默认字体大小
	3. font-family 字体族 设置字体的样式 
		可选值 ： serif 衬线字体 ； sans-serif 非衬线字体 ； monospace 等宽字体 ；
		font-family 可同时指定多个字体 多个字体间用 , 隔开 字体生效时优先使用第一个 往后以此类推
	4. font-weight 字重 设置字体的加粗
		可选值 ： normal 默认值； bold 加粗； 
	5. font-style 字体风格
		可选值 ： normal 默认值； italic 斜体；
		
.demo1{
            border: 2px red solid;
            /* 设置demo1 字体颜色 */
            color: skyblue;
            /* 设置demo1 字体大小 */
            font-size: 18px;
            /* 设置demo1 字体样式 非衬线字体*/
            font-family: sans-serif;
        }
        .demo2{
            border: 2px red solid;
            color: green;
            font-size: 1em;
            font-family: monospace;
        }
        .demo3{
            border: 2px red solid;
            color: aqua;
            font-size: 2rem;
            font-family: serif;
        }


字体的简写格式
font 可以设置字体相关的所有属性 语法： font: 字体大小/行高 字体族 行高 可以省略不写 如果不写使用默认值
	 p{
            border: 2px red solid;
            font: 30px/40px monospace;
            /* 将字体设置为 大小为 30px 行高为40px 字体族为等宽字体 */
            font: italic bold 30px serif;
            /* 将字体设置为 斜体 粗体 大小为30px 行高默认 字体族为衬线字体 */
        }

2.使用自定义字体格式

在 <style> 标签中导入自已的字体样式资源

	@font-face{
        /* 指定字体在网页中使用时的名字 */
        font-family: '自己起的字体名字';
        src: url('字体所在的路径') format("truetype")
}
.demo4{
            border: 2px red solid;
            color: black;
            font-size: 1em;
            /* 使用自定义字体样式 */
            font-family: myfont;
        }

3.矢量图标–Awesome的使用

1.下载 https://fontawesome.com/
            2.解压
            3.将css和webfonts移动到项目中
            4.将all.css引入到网页中
            5.使用图标字体
                - 直接通过类名来使用图标字体
                    class="fas fa-bell"
                    class="fab fa-accessible-icon"
PS ： class 中 只有fas 和 fab 是免费的 当图标字体加载不出来的应在fas 和 fab 中切换尝试

<link rel="stylesheet" href="./fa/css/all.css">

方法1：
 <!-- 通过直接在标签类名使用图标 -->
    <i class="fas fa-spider" style="font-size: 40px; color: red;"></i>

方法2：
 <!-- 通过使用实体编码使用图标  &#x实体编码; -->
    <i class="fas" style="font-size: 25px;">&#xf6be;</i>

方法3：在伪元素选择器中使用图标
p::before{
            /* 使用 \ 加实体编码 作为图标的表示 */
            content: '\f0a4';
            font-size: 30px;
            /* 必须指定图标族名 Font Awesome 5 Brands/Free 二选一 */
            font-family: 'Font Awesome 5 Free';
        }
        p::after{
            content: '\f0a5';
            font-size: 30px;
            font-family: 'Font Awesome 5 Free';
        }
    <p>
        好一似今朝沦落人，且回首往昔荣华日。 --鼻子
    </p>

4.阿里字体库使用–iconfont

1.在阿里图标库官网选择需要的图标 添加至项目 下载项目并解压
2.将文件放入网页静态资源中 通过 link 引入样式 iconfont.css 文件
3.使用方法和Awesome基本一致 在标签中使用时类名均为 iconfont
	<link rel="stylesheet" href="./MyiconFont/iconfont.css">

方法1：
	<!-- 使用标签类名使用图标 -->
    <i class="iconfont icon-a-1"></i>

方法2：
    <!-- 使用实体编码使用图标 -->
    <i class="iconfont">&#xe733;</i>

方法3：
 p::before{
     		/* 转义编码为实体编码&#x 后的字符 */
            content: '\e73f';
            font-family: 'iconfont';
            font-size: 100px;
        }
 	<p>

    </p>

5.行高

行高指文字占有的实际高度 可通过 line-height 指定行高
行高可以直接指定一个大小（px em）
                        也可以直接为行高设置一个整数
                            如果是一个整数的话，行高将会是字体的指定的倍数
 行间距 = 行高 - 字体大小
 字体框 - 字体框就是字体存在的格子，设置font-size实际上就是在设置字体框的高度
 			行高会在字体框的上下平均分配

.demo{
            /* 设置字体大小 */
            font-size: 50px;
            /* 设置边框 */
            border: 2px red solid;
            /* 设置行高 */
            line-height: 50px;
            /* 1.此时文字和行高一致 若此时文字的行高小于字体大小的话 文字将叠在一起 */
            line-height: 60px;
            /* 2.此时行高大于字体大小 即实际上文字的高度小于文字所在行的高度 此时可见文本行间距较大 */
        }

6.文本的样式

1.文本水平，垂直对齐

text-align 属性可设置文本的水平对齐方式
	可选值：left 左对齐；right 右对齐；center 居中对齐； justify 两端对齐；
.demo{
            border: 2px red solid;
            font: 30px monospace;
            text-align: center;
            /* 1.居中对齐 */
            text-align: left;
            /* 2.居左对齐 */
            text-align: justify;
            /* 3.两端对齐 */
            text-align: right;
            /* 4.居右对齐 */
        }

vertical-align 属性设置元素的垂直对齐方式
	可选值： baseline 默认值 基线对齐
			top 顶部对齐
			bottom 底部对齐
			middle 居中对齐

		.demo1{
            font: 50px monospace;
            border: 2px red solid;
        }
        span{
            border: 1px blue solid;
            font: 30px serif;
            vertical-align: baseline;
            /* 1.设置基线对齐 默认值 */
            vertical-align: top;
            /* 2.设置垂直顶部对齐 此时span元素在行间以父元素的顶部对齐 */
            vertical-align: bottom;
            /* 3.设置垂直底部对齐 此时span 元素在行间以父元素的底部对齐 底部对齐低于基线对齐 */
            vertical-align: middle;
            /* 4.设置垂直居中对齐 此时span元素在行间以父元素中的x的中心点为基准对齐*/
        }

        上善若水，水善利万物而不争。x<span>Hellox督导室覅女卫</span> --老子·道经·第八章

2.文本修饰

text-decoration 设置文本修饰
	可选值：none 什么都没有； underline 下划线； line-through 删除线； overline 上划线；

 .demo1{
            font: 30px monospace;
            text-decoration: none;
            /* 1.text-decoration : none 此时文本无效果 */
            text-decoration: line-through;
            /* 2.删除线效果 */
            text-decoration: underline;
            /* 3.下划线效果 */
            text-decoration: overline;
            /* 4上划线效果 */
        }
<div class="demo1">
        凡事都有一个与一望而知不同的真相 --约翰·勒卡雷
    </div>

white-space 设置网页如何处理空白
	可选值： normal 正常 默认值； nowrap 不换行； pre 保留空白；

 .demo2{
            font: 30px sans-serif;
            white-space: normal;
            /* 1.normal 正常 此时网页解析时 多个空格解析成一个空格 */
            white-space: nowrap;
            /* 2.nowrap 不换行 此时网页解析时文本段不换行 */
            white-space: pre;
            /* 3.pre 保留空白 此时网页解析时按照编辑器里文本的格式解析*/
        }
