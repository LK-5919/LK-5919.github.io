运算符

通过运算符可以对一个或多个值进行运算，并获取运算结果；

一元运算符

一元 : 只需要一个操作数；
" + "号，正号不会对数字（number）产生任何影响；
" - "号，负号可以对数字进行算术取反；
	对于非Number类型的数据来说，使用一元运算符时，会将其转换成number类型 --- 原理和使用Number()一致；

自增和自减

自增 ++ ；
	自增分为两种，前++和后++，这两者都会使原变量的值自增1；但是 i++ 和 ++i 的值是不一样的；
    var i = 1;
	console.log("i++ = " + i++);  //输出 i++ = 1;
	i = 1;
	console.log("++i = " + ++i);  //输出 ++i = 2；
	可知 i++ 等于原变量的值，但是 ++i等于原变量自增1后的值；
    即若 a = i++ ;  //即是将 a = i ；然后 i = i + 1； 即后加加是先使用原变量的值，之后原变量的值再自增1；
		a = ++i ; //即是将 i = i + 1 ； 然后 a = i ；即前加加是先原变量的值自增1，之后再将使用自增后的值；
		理解： 后加加（i++）里变量在前，+号在后，所以先使用原变量，之后原变量再自增；
				前加加（++i）里+号在前，变量在后，所以先变量自增1，之后再使用变量；

自减 -- ；
自减也分为两种 ，前-- 和 后-- ，这两者都会是原变量的值自减1， 但是 i-- 和 --i的值是不一样的；
	具体使用和 ++ 一致；

算术运算符

1.加法 "+" ：如果对两个字符串进行加法运算，会进行字符串的拼接，任何值和字符串相加都会想转换成字符串，然后进行字符串的拼接操作；
   	可以用利用这一特点进行任何数据类型进行字符串的隐式转换 ---- 实际上也是调用String()函数返回；
    	var a = 123;
 		console.log(a + '');  //输出字符串的“123”

2."-" , "*" , "/" , 任何值 做减法 ，乘法， 除法 运算时都会自动转换成Number类型；
   	可以利用这点将任何数据类型进行Number的隐式转换；
       var a = "123";
	console.log(a-0);  //输出number类型的 123

3.模运算 "%" : 求余数运算；

逻辑运算符

&& --- 与
	两个值中只要有一个false时就返回false，只有两个值为true时才返回true，如果第一个值为false则直接返回false不会往下检查；

|| --- 或
	两个值中只要有一个true时就返回true，只有两个值均为false时才返回false，如果第一个值为true时直接返回true不会继续往下检查；
    
! --- 非
	对布尔值进行取反运算；
    	如果对非布尔值进行取反，则会将其转换为布尔值然后再取反；
        可以利用对非布尔值去两次反，这样就可以实现将其他任意的数据类型进行布尔值的隐式转换；

赋值运算符

可以将符号右侧的值赋给符号左侧的变量；

关系运行符

通过关系运算符可以比较两个值之间ed大小关系；
对于非数值使用关系运算符比较的情况：会将其转换成数字然后进行比较；
	如果比较的两侧都是字符串的时候，不会将其转换成数字比较，而是依次比较字符串中每一个字符的Unicode编码，一个一个字符进行比较，相比较的字符编码大，则该字符串就大，不会继续往下比较，若一个字符串是另一个字符串的前缀，则字符串长度长的大；

编码

在字符串中使用转义字符输入Unicode编码 --- \u字符编码 -- \u2620 ;
在网页中使用Unicode编码是使用 &#字符编码 编码需要转换成10进制；

相等运算符

使用 == 运算符来比较两侧的值是否相等；
使用 != 运算符来比较两侧的值是否不相等；

== 和 != 在比较不同的类型的值时，会自动将其转换成相同类型进行转换，之后再比较；
console.log( null == 0); //输出false ；
null 比较特殊，比较运算符  >=   <=   >   < 会对其进行数据类型转换； 相等运算符 == != 不会对其进行数据类型转换。

undefined衍生自 null ，所以这两个值做相等判断时，返回true；
console.log( undefined == null );  //返回true；

NaN不和任何值相等 ， 包括它本身；
console.log( NaN == NaN );   //返回 false；
要判断 变量的值是否为 NaN，可使用 isNaN() 函数；
console.log(isNaN(NaN));  //返回 true；

使用 === 和 !== 来判断两侧的值是否相等，但是它不会进行类型转换，若比较的数据两者类型不同，则直接判断为不相等；

条件运算符（三元运算符）

语法：  条件表达式?语句1:语句2;
	先执行条件表达式，若表达式值为true，则执行语句1，否则执行语句2；
    如果条件表达式执行后值不是布尔值，会将其转换成布尔值后再执行后面的语句；
   var a = 1;
	var b = "hello";
    a > b ? alert("a大") : alert("b大");  //执行语句2；
