1.表格

使用 table 标签 在网页中创建表格
	tr 标签表示表格中的一行
	td 标签表示表格中的一个单元格
		可在table标签中设置表格的边框属性 默认为没有边框
		table是块元素 但是其默认的宽度不会是占满一行 而是由内容撑开 其他的由浏览器自动调整margin-right

<!-- 创建一个表格 -->
    <!-- 在table标签中 设置border边框属性 -->
    <table border="1px" >
        <tr>
            <td>学号</td>
            <td>姓名</td>
            <td>年龄</td>
        </tr>
        <tr>
            <td>001</td>
            <td>gbfdhsj</td>
            <!-- 使用rowspan属性合并行单元格 -->
            <td  rowspan="2">man</td>
        </tr>
        <tr>
            <td>001</td>
            <td>Kods</td>
            
        </tr>
        <tr>
            <td>001</td>
            <!-- 使用colspan属性设置列单元格合并 -->
            <td colspan="2">Kods</td>
        </tr>
    </table>


表格可以分为三个部分 thead -- 头部 ； tbody -- 主体 ； tfoot -- 底部 ；
	在行中使用th来表示表格的首行的单元格 此时用于表示表格的列值名称 th表示的单元格文字默认加粗

<table border="1px" width="400px" style="margin:20px auto;">
        <thead>
            <tr>
                <th>学校</th>
                <th>名字</th>
                <th>分数</th>
                <th>专业</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>惠州学院</td>
                <td>hjihu</td>
                <td>111</td>
                <td>计算机</td>
            </tr>
            <tr>
                <td>惠州学院</td>
                <td>hjihu</td>
                <td>111</td>
                <td>计算机</td>
            </tr>
            <tr>
                <td>惠州学院</td>
                <td>hjihu</td>
                <td>111</td>
                <td>计算机</td>
            </tr>
            <tr>
                <td>惠州学院</td>
                <td>hjihu</td>
                <td>111</td>
                <td>计算机</td>
            </tr>
        </tbody>
    </table>

border-spacing: 指定边框之间的距离 此时指两边框间的距离 不是边框重合
border-spacing: 0px; 两边框相邻间隔为0
border-collapse: collapse; 设置边框的合并 此时指两边框合并为一条边框 

 table{
            width: 500px;
            margin: 30px auto;
            border: 1px solid black;
            /* 此时在css文件中设置table的边框样式只有外边框 单元格此时没有边框 */
            border-spacing: 0px;
            /* 2.利用border-spacing属性设置边框间的间隔为0px
                此时的边框为2px
            */
            border-collapse: collapse;
            /* 3.设置边框重合 此时边框只有1px*/
        }
        td{
            border: 1px solid black;
            /* 4.此时给每一个单元格设置外边框 加上table的外边框此时呈现双边框效果 */
            text-align: center;
            /* 5.设置单元格中文字居中 */
            /* 单元格中的元素默认垂直居中 */
            /* vertical-align: top; */
        }
        table  tr:nth-child(odd){
            background-color: #bfa;
            /* 在写代码时若我们没有使用tbody 浏览器在解析时也会自动加上tbody' */
            /* 设置奇数行背景颜色 */
        }
        table tr:nth-child(even){
            background-color: aliceblue;
        }

2.表单

在网页中使用 form 标签创建表单
	在action中值为处理表单收集到的信息发送的服务器地址
	表单中输入的信息可选input标签 通过选择type类型来设置输入类型
		text -- 文本
		password -- 密码
		radio -- 单选框（单选框需要使用同一组name值来限定）
		checkbox -- 多选框
		color -- 颜色选择器
	表单元素必须指定name属性
	readonly 将表单项设置为只读，数据会提交
	disabled 将表单项设置为禁用，数据不会提交
    autofocus 设置表单项自动获取焦点

autocomplete="off" 关闭自动补全
		
  <form action="target.html">
        <div class="title">
            <h1>用户注册</h1>
        </div>
        <div class="info-input">
            <label for="username">姓名</label>
            <input type="text" id="username" name="username" placeholder="请输入用户名" autocomplete="off" value="1212" readonly autofocus>
        </div>
        <div class="info-input">
            <label for="password">密码</label>
            <input type="password" id="password" name="password" placeholder="请输入密码" value="1212" disabled>
        </div>
        <div>
            <label for="sex">性别</label>
            <input type="radio" value="1" name="sex" id="sex" checked>男
            <input type="radio" id="sex" value="2" name="sex">女
        </div>
        <div>
            <label for="major">专业</label>
            <select id="major" name="major">
                <option value="1">计算机</option>
                <option value="2" selected>数学</option>
            </select>
        </div>
        <div>
            <label for="hobby">爱好</label>
            <input type="checkbox" name="bobby" id="hobby" value="1">篮球
            <input type="checkbox" name="bobby" id="hobby" value="2">足球
            <input type="checkbox" name="bobby" id="hobby" value="3">跑步
        </div>
        <input type="color" name="" id="">
        <input type="submit" value="注册">
    </form>
