流程控制语句

JS中代码按照从上到下的顺序执行，通过流程控制语句可以控制程序的执行流程，流程控制语句分为 ：

​ （1）条件判断语句；

​ （2）条件分支语句；

​ （3）循环语句；

条件判断语句 — if语句

语法一： if(条件表达式){
    语句；
    ....
}
if语句在执行时，一旦括号里的条件表达式为true时，就执行其下的代码块；

语法二： if(条件表达式){
    		语句；
    		...
		}else if(条件表达式){
            语句；
            ....
        }else{
            语句;
            ....
        }
  当执行该语句时，会从上依次判断if后面的条件表达式，在该语句中，只有一个代码块会执行，一旦进入某个代码块，执行完毕后将结束整个if语句，不会继续往下判断if条件表达式；当所有的if条件表达式均为false时，将进入else代码块；

条件分支语句 — switch语句

语法 ： 
	switch(条件表达式){
        case 表达式1:
            语句；
            ...
            break;
        case 表达式2::
        	语句；
            ...
            break;
            
            ....
            
         default:
            语句；
            ...
            break;
    }

	在执行时会依次将case后面的表达式与switch后的条件表达式进行 全等比较（===） ,只有比较结果为true时才会执行当前case后的语句；
	在case语句后的代码块中最后需要写入 break 关键字，否则当前case执行完以后会继续执行后面的case语句（不用判断直接执行）；
	若所有的case语句的表达式与switch中的条件表达式比较均为false，则执行default语句（也可以不书写default语句，此时将结束switch语句）；
    
    if语句和switch语句功能上两者可互相代替；

循环语句 — while循环

循环语句： 通过循环语句可以反复执行一段代码多次；

while循环  语法：
	while(条件表达式){
          语句；
          ...
          }
     
     while语句在执行时先对条件表达式进行求值判断，若条件表达式为true，则进入循环体，循环体代码块执行完毕以后再判断条件表达式是否成立，是否需要进入下一次循环；否则终止循环；
     
   do...while 循环 语法
     do{
         语句；
         ...
     }while(条件表达式)
         
     do...while语句在执行时会先执行循环体，循环体执行完以后在对while的条件表达式进行判断，结果为true则继续执行循环体否则结束循环；
         
     这两个循环语句功能类似，while语句是先判断再循环，do...while语句是先执行再判断，所以do...while语句至少执行一次；

循环语句 — for循环

for 循环 语法：
	for(初始化表达式;条件表达式;更新表达式){
        语句;
        ...
    }
        
    for循环执行流程：
    	1.执行初始化表达式，初始化变量（初始化表达式只会执行一次）；
        2.执行条件表达式，判断是否进入循环；
     	3.执行更新表达式，更像表达式执行完以后在继续执行第二步；
        
        for循环的三个部分都可以省略，也可以写在外部，如果在for循环中不写任何表达式只写两个 ;  此时循环会是一个死循环会一直执行下去；
        
 for循环可以嵌套使用；

break和continue

	break关键字可以用来退出switch或循环语句，但是不能单在if语句中只用，break关键字会立即终止离它最近的那个循环语句；
   	可以为循环语句创建一个label，来标识当前的循环；
    	label:循环语句
        在使用break语句时，可以在break后跟着一个label，这样break就会结束指定的循环而不是最近的循环；
        
     continue关键字可以跳过本次循环，同样continue也是默认只会离它最近的循环起作用；
     
     outer:
			for(var i=0 ; i<5 ; i++){
				console.log("@外层循环"+i)
				for(var j=0 ; j<5; j++){
					break outer; //这里的break指定了结束外部的循环体
					console.log("内层循环:"+j);
				}
			}

 			/*
			 * 对于成绩大于60分的，输出'合格'。低于60分的，输出'不合格'
             */
		  var score = 61;
            switch (true){
                case score >= 60 :
                    console.log("合格");
                    break;
                default :
                    console.log("不合格");
            }
            console.log(typeof !!score);
            console.log(!!score);

输出：
合格
boolean
true

JS的switch中表达式可以是boolean值，但是java中不可以！！！在这里若把switch的表达式的 true 改为 score 则将只能执行default，因为switch中的表达式并不会自动进行类型转换，而是将switch的表达式与case的表达式进行全等比较（===）；
