010-背景-颜色渐变

2022-03-05 HTML-CSS学习 1.9k 字
WEB前端

1.背景

background-color 设置背景颜色

background-image 设置背景图片
	可以同时设置背景颜色和背景图片 此时背景颜色会成为背景图片的背景色
		1.如果背景图片小于当前的元素大小 背景图片会自动在元素中平铺
		2.如果背景图大于元素 将会溢出父元素 溢出元素的背景无法显示
		3.若背景图片和元素一样大 则会直接正常显示

 background-repeat 用来设置背景的重复方式
         可选值：
                repeat 默认值 ， 背景会沿着x轴 y轴双方向重复
                repeat-x 沿着x轴方向重复
                repeat-y 沿着y轴方向重复
                no-repeat 背景图片不重复

 background-position 用来设置背景图片的位置
                    设置方式：
                        通过 top left right bottom center 几个表示方位的词来设置背景图片的位置
                            使用方位词时必须要同时指定两个值，如果只写一个则第二个默认就是center

                        通过偏移量来指定背景图片的位置：
                            水平方向的偏移量 垂直方向变量

.demo{
            width: 300px;
            height: 300px;
            background-color: #bfa;

            /* 设置背景图片 */
            background-image: url('./img/1.png');
            /* 1.原背景图片 大小为  128* 128 此时背景图小于元素
                此时背景图默认在元素中重复平铺展示
            */

            /* width: 100px;
            height: 100px; */
            /* 2.将元素大小修改成小于背景图 此时溢出元素的背景无法显示 */

            /* width: 128px;
            height: 128px; */
            /* 3.此时元素与背景图大小一致 正常显示 */

            background-repeat: repeat;
            /* 4.设置背景图的重复方式 此时背景图小于元素 
            且重复方式为 repeat默认值 此时背景图水平垂直方向上均重复*/

            /* background-repeat: repeat-x; */
            /* 5.设置背景图重复方式为水平重复 此时垂直方向上无重复 */

            /* background-repeat: repeat-y; */
            /* 6.设置背景图垂直重复 此时背景图水平方向上无重复 */

            background-repeat: no-repeat;
            /* 7.设置背景图水平垂直方向上均无重复 */

            background-position: 0 0;
            /* 8.设置背景图偏移量来设置背景图的位置 此时 0 0 为默认值 
                为父元素文档流中的位置 元素的内边距区开始计算
            */

            background-position: 10px 20px;
            /* 9.第一个值为水平方向上的偏移（正值时图片向右移动）
                第二个值为垂直方向上的偏移（正值时图片向下移动）
            */

            background-position: center center;
            /* 10. 通过方位词来设置背景图的位置 */

        }

background-clip 设置背景范围
	可选值 : border-box 默认值 此时背景图会出现在边框的下边
			padding-box 此时背景不会出现在边框下边 但会出现在内容区和内边距区
			content-box 此时背景只会出现在内容区

.demo{
            width: 200px;
            height: 200px;
            background-color: #bfa;

            border: 10px transparent solid;
            /* 给元素添加10px 的透明边框  */
            padding: 10px;
            /* 给元素添加 10 px 的内边距区 */
            background-clip: border-box;
            /* 1.此时的background-clip 值为border-box 默认值 此时背景会出现在边框区 和 内边距区*/
            background-clip: padding-box;
            /* 2.设置background-clip 值为 padding-box 
            此时背景颜色不会出现在边框区 此时元素的外边框因背景透明不可见 */
            background-clip: content-box;
            /* 3.设置背景颜色只出现在内容区 此时内边距区因背景透明不可见 */

        }

backgroung-origin 设置背景图偏移量（background-position）的计算原点
	可选值： padding-box 默认值 此时计算原点从内边距区处开始计算
			content-box 背景图偏移量计算原点从内容区开始
			border-box 背景图偏移量的计算原点从边框处开始计算

background-size 设置背景图的大小 第一个值表示宽 第二个值表示高
	若只写一个值 则默认第二个值是auto
	cover --- 图片的比例不变 将元素铺满
	contain --- 图片比例不变 将图片在元素中完整显示

background-attachment 设置背景是否会跟随元素移动
	可选值 : scroll 默认值此时背景图会跟随元素移动
			fixed 背景图会固定在页面中 不会跟随元素移动

.demo1{
            width: 1920px;
            height: 1000px;
            background-color: #bfa;
            background-image: url('./img/2.jpg');
            /* 此时背景图远大于元素 溢出部分无法显示 */

            background-size: 500px;
            /* 1.通过设置background-size 设置背景图的宽为500px 此时高等比例缩小
            此时背景图在元素内平铺 效果和 contain 一致
            */

            background-size: cover;
            /* 2.图片的比例不变 将元素铺满 */
            /* background-size: contain; */
            /* 3.图片的比例不变 将图片在元素中完整显示 可能会出现重复平铺 */
            background-repeat: no-repeat;
            /* 4.设置背景图不重复平铺 */

            border: 10px solid red;
            padding: 20px;
            /* 设置外边框个内边距 此时背景图延伸到内边距区 */

            background-origin: padding-box;
            /* 5.设置背景图的偏移计算原地为内边距区 */
            /* background-position: 10px 10px; */
            
            background-origin: content-box;
            /* 6.设置背景图的偏移计算原点为内容区 */

            background-origin: border-box;
            /* 7.设置背景图偏移量计算原点为外边框区 */

            background-attachment: scroll;
            /* 8.设置背景图跟随元素移动 scroll 为默认值 */

            background-attachment: fixed;
            /* 9.设置背景图以原图大小固定在在页面中元素文档流的位置 
                溢出的背景不可见 滑动页面元素移动时背景不会动
                想象是移动画框 画固定在原地
            */
        }

2.背景色渐变

1.水平垂直线性渐变

渐变 ： 一个颜色向其他颜色过渡的效果 
PS： 渐变是图片！！ 需通过background-image 来设置
	linear-gradient 线性渐变 颜色沿着一条直线发生变化
	线性渐变的开头 可以指定一个渐变的方向
		to left ； to right ； to bottom ； to top ； to top left ...
		deg --- 度数 ； turn --- 圈数 ；
	渐变可以同时指定多个颜色 多个颜色情况下平均分布；
		可以手动指定渐变的分布情况；

.box2{
            width: 200px;
            height: 200px;
            background-image: linear-gradient(red,green);
            /* 设置线性渐变 从红色到绿色 默认情况上是 从上到下渐变 各占一半分布 */

            background-image: linear-gradient(to right,red,yellow);
            /* 设置从左到右的线性渐变 从红色到绿色 */

            background-image: linear-gradient(to left, red,yellow 50px,gold 100px ,orange 150px,grey 200px);
            /* 设置从左到右由红色开始渐变到黄色占50px 
                再由黄色渐变到金色 占50px
                再由金色渐变到橘色 占50px
                再由橘色渐变到灰色 占50px
            */

            background-image: linear-gradient(90deg, red,yellow );
            /* 0度 时为从下到上渐变 90deg为顺时针旋转90度 此时为由红色到黄色 从左到右水平线性渐变*/

            background-image: linear-gradient(0.25turn,red,yellow);
            /* 0turn为从下到上渐变  0.25turn相当于顺时针旋转90度*/
        }

2.径向渐变

background-image：radial-gradient（） 可设置径向渐变（放射性效果）
	默认情况下径向渐变的形状根据元素的形状来计算 正方形 --- 圆形； 长方形 --- 椭圆形；
	也可以手动指定径向渐变的大小
		1.circle ； 2.ellipse ； 
	指定渐变的位置
		radial-gradient（大小 at 位置 ， 颜色 位置， 颜色 位置，颜色 位置）
				大小：circle 圆形；ellipse 椭圆形；closest-side 近边；closest-corner 近角； 
					farthest-side 远边； farthest-corner 远角；
 .box2{
            width: 200px;
            height: 200px;
            background-image: radial-gradient(circle at 50px 50px,red,yellow);
            /* 设置径向渐变  */
        }
详见参考官方文档  ： https://developer.mozilla.org/zh-CN/docs/Web/CSS/gradient/radial-gradient()
