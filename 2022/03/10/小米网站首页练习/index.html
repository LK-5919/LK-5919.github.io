小米网站首页练习

2022-03-10 HTML-CSS学习 1.4k 字
WEB前端

顶部导航栏

	小米首页的布局都是在一个外部容器宽占100%中放入一个内部容器布局，其中内部大容器的宽度都是一致 1226px ，这样可以通过设置水平的margin为auto达到内部元素水平居中；
	
	导航栏布局较简单：
	内部容器为一个大的div
		里面划分为3个ul
			左边的ul 开启 float:left
			右边的两个ul均开启 float:right; (注意float的先后顺序)
		每一个ul里的li元素均开启float:left使li元素可排列在同一排
	在开发中应设置li中的a标签转换为 block 元素 以使其高度与外部的div一致 增大a标签的可点击范围；
	

导航栏设置主要难度在弹出层的设置：在鼠标移入 下载app 的li标签时 其下部应该展示下载的二维码，具体设置如下：
	
1.应该设置弹出层的div放在a标签里面，这样才能确保在鼠标移入a标签时弹出层出现且在鼠标移入弹出层时a标签依旧保持hover效果，而且网页中的弹出层是可以点击的；
			<li>
                    <a href="javascript:;" class="app">下载app
                        <div class="qrcode">
                            <img src="./mi_images/download.png">
                            <span>小米商城APP</span>
                        </div>
                    </a>
                </li>

2.a标签开启相对定位（此时弹出层div为其子元素），所以弹出层div要开启绝对定位来设置弹出的位置；

3.弹出层的过渡效果通过设置弹出层div的高度开始为0，当鼠标移入a标签时，弹出层div高度逐渐增大到正常高度，使用transition属性设置，transition属性的开启设置在要变化的元素身上！！！

4.购物车的弹出层实现方法和上述相似，我这里的购物车是一个单独的ul，而且这个ul里面只有一个li，所以我将弹出层和a标签放置在同级，为兄弟关系；
<!-- 右侧购物车 -->
            <ul class="shop-cart">
                <li>
                    <a href="javascript:;">
                        <i class="fas fa-shopping-cart"></i>
                        购物车（0）
                    </a>
                    <div class="mycart">
                        <span>购物车中还没有商品，赶紧选购吧！</span>
                    </div>
                </li>
            </ul>

5.弹出层的小三角箭头：通过绝对定位这是一个宽高均为0，但是有边框的div，再将不需要的边框设为透明可以实现小三角；
<style>
.app::after{
    display: none;
    content: '';
    border: 8px solid transparent;
    border-bottom-color: white;
    height: 0;
    width: 0;
    /* 要设置宽度才能实现居中 */
    position: absolute;
    bottom: 0;
    left: 0;
    right: 0;
    margin: auto;
}
</style>

注意要设置弹出层的层级 z-index；

logo栏

logo栏布局也是一个大的div中划分为三个部分，其中左侧div中是两个a标签，中间的是商品分类，采用ul布局，右侧的搜索框中是一个输入框和按钮；

主要的难度是logo区图片的切换：
	外层的div实现设置好大小，且开启overflow：hidden，以此隐藏div外的图片；
	外层div开启相对定位，内部的两个a标签开启绝对定位，当鼠标移入div时，通过改变图片的left偏移量来实现图片的平移，通过给两个a标签开启transition属性实现过渡效果；

中间的ul布局中的li中的a标签应该同父元素高度一致，ul中的最后一个li放的是弹出层，弹出层的绝对定位通过外部容器的相对定位来定位，以此实现弹出层的宽度为100%；

ul中的第一个li为隐藏元素，该元素里面有一个纵向的导航栏，该导航栏的绝对定位相对于这第一个隐藏元素li的相对定位；

录播图区块

轮播图区块设置主要在切换按钮的雪碧图和右下角的小圆点按钮

雪碧图和小圆点均是开启绝对定位，相对于放入img元素的父元素的相对定位；

伪元素边框

该元素的边框均是通过给div中的li元素添加伪元素；
<style>
     .home-shop-wrapper .shortcut li::before{
        content: '';
        width: 64px;
        height: 1px;
        background-color: #665e57;
        /* background-color: red; */
        position: absolute;
        left: 0;
        right: 0;
        top: 0;
        margin: 0 auto
    }
    .home-shop-wrapper .shortcut li::after{
        content: '';
        width: 1px;
        height: 70px;
        background-color: #665e57;
        /* background-color: red; */
        position: absolute;
        left: 0;
        top: 0;
        bottom: 0;
        margin: auto 0;
    }
</style>

固定工具栏

使用固定定位一个ul列表；其右侧的弹出层相对于它的每一个li元素定位，当鼠标移入li时，a标签的下一个兄弟元素展示；

广告卡片展示

卡片均开启相对定位，当鼠标移入时修改卡片的top属性，加上阴影box-shadow实现卡片效果；

总结

/*
	这次的小米首页布局历时了几天才完成，虽然内容不多，但是算得上是第一次完成比较完整的网页布局，开发网页都需要先考虑好元素的摆放布局，之后再考虑样式问题；
	这个网页使用了相对布局，绝对布局，固定布局，也是第一次接触到transition过渡属性；
	在练习时发现了类选择器的优先级问题，写的详细的类选择器优先级较高；
	总的来说这次练习再次复习了一下之前所学的知识，对布局的等式有了进一步理解；
*/
