004-CSS选择器

1.CSS语法简介

<!--
	方式1 内联样式/行内样式：在标签内通过style属性设置元素样式
		内联样式：属性:值; 使用 ; 来间割属性
		该方式不能实现样式复用
-->
<p style="color:green; font-size:20px;">
    这里是一段p元素文字
</p>

!!!注意在实际开发中不要轻易使用内联样式

<!-- 
	方式2 内部样式表：在head标签内写style标签
			语法 样式选择器{color: red;}   
	方式3 外部样式表：通过引入外部css文件
			在head中通过link标签引入
-->
<link rel="stylesheet" href="这是css文件路径">

在实际开发中推荐使用方式3 css文件可实现复用

css文件中的注释符号为  /*  这是注释 */

2.CSS选择器

1.常用选择器

（1）元素选择器：根据标签名选中指定元素 页面中所有该元素的标签都有效
	语法：标签名{}  如p{}，h1{} ，h2{} body{}
（2）id选择器： 根据标签的id选中元素
	语法：#id名{} 如 #box{}， #big{}
		使用id选择器时 不提倡id重复 即id选择器只选中一个标签元素
			同id名是 目前的id选择器都将生效 效果和类选择器相同 但不推荐使用相同id
（3）类选择器：使用类名选择一组相同类名的元素
	语法：.className{},如 .类名{} 
（4）通配选择器：选中页面所有元素
	语法：*{} 


p{
    这是元素选择器
    color: red;
}
#red{
     这是id选择器
     color: red;
}
.box{
    这是类选择器
    color: red;
}
*{
    这是通配选择器
    效果和元素选择器 body{} 相同 选中所有元素
}

2.复合选择器

（1）交集选择器：选中同时复合多个条件的元素
	语法：选择器1选择器2选择器3{}  
		交集选择器中如果有元素选择器 需使用元素选择器开头
div.box{
    这是一个交集选择器 选择一个类名为box的div元素
    color:red;
}

	（2）并集选择器：同时选择多个选择器对应的元素
     语法：选择器1,选择器2,选择器3{}
div,span,h1{
    这是一个并集选择器 同时选中div，span，h1 三种标签
    color: red;
}

3.关系选择器

父元素
          - 直接包含子元素的元素叫做父元素
      子元素
          - 直接被父元素包含的元素是子元素
      祖先元素
          - 直接或间接包含后代元素的元素叫做祖先元素
          - 一个元素的父元素也是它的祖先元素
      后代元素
          - 直接或间接被祖先元素包含的元素叫做后代元素
          - 子元素也是后代元素
      兄弟元素
          - 拥有相同父元素的元素是兄弟元素

1.子元素选择器

（1）子元素选择器：选中指定父元素中的指定子元素
			语法：父元素 > 子元素
div.box > span {
    /* 选中类名为box的div中的所有span子元素 只选中子元素*/
    color: red;span {
            color: aqua;
            /* 1.给所有span元素设置颜色 */
            }
            .demo1 > span{
            color: red;
            /* 2.选择demo1 中下属的直接子元素（不包括子元素的后代） 可见只有demo1
                下的子元素才有效果 demo1的孙子元素及其以后未被选中！
            */
          }
}

<div class="demo1">
        <span>Span #1, in the div.
            <span>Span #2, in the span that's in the div.</span>
          </span>
          <span>我是demo中的第三个span子元素</span>
    </div>

4.属性选择器

属性选择器：选择指定属性的标签元素
		语法：（1）[属性名=属性值] 选择含有指定属性和指定元素值的标签
如 p[name="box"]{
    选择name值等于“box”的p标签
}
		（2）选择属性值以指定值开头的元素  [属性名^=属性名]
如 p[name ^= "box"]{
    选择name值以“box”开头的p标签
}
		（3）选择属性值以指定值结尾的元素  [属性名 $= 属性值]
如p[name $= "box"]{
    选择name值以“box”结尾的元素的p标签
}
		（4）选择属性值中含有某值的元素  [属性名 *= 属性值]
如 p[name *= "box"]{
    选择name值中含有“box”的p标签
}

5.伪类选择器

伪类选择器：伪类 （不存在的类 特殊的类 伪类用来描述一个元素的特殊状态）
						如：第一个子元素 被点击的子元素等
						语法： : 开头
			   :first-child 第一个子元素
                :last-child 最后一个子元素
                :nth-child() 选中第n个子元素
                    特殊值：
                        n 第n个 n的范围0到正无穷
                        2n 或 even 表示选中偶数位的元素
                        2n+1 或 odd 表示选中奇数位的元素

                    - 以上这些伪类都是根据所有的子元素进行排序

                :first-of-type
                :last-of-type
                :nth-of-type()
                    - 这几个伪类的功能和上述的类似，不通点是他们是在同类型元素中进行排序

            - :not() 否定伪类
                - 将符合条件的元素从选择器中去除

 /* 使用伪类选择器 选择 类名为demo 的ul 中的第一个li子元素 */
        .demo > li:first-child{
            /* 这里选择第一个子元素 */
            color: yellow;
            font-size: 20px;
        }
        /* 使用伪类选择器 选择 类名为demo 的ul 中的最后一个li子元素 */
        .demo > li:last-child{
            /* 这里选择最后一个子元素 */
            color: skyblue;
            font-size: 50px;
        }
        /* 使用伪类选择器 选择 类名为demo 的ul 中的第奇数个li子元素 */
        .demo > li:nth-child(2n+1){
            /* 这里选择第奇数个子元素 */
            background-color: red;
        }
          /* 使用伪类选择器 选择 类名为demo 的ul 中的第偶数个li子元素 */
          .demo > li:nth-child(2n){
            /* 这里选择第偶数个子元素 */
            background-color: gray;
        }
          /* 使用伪类选择器 选择 类名为demo 的ul 中的第一个span子元素 */
          .demo > span:first-of-type{
            background-color: orange;
        }
        /* 使用伪类选择器 选择 类名为demo 的ul 中的除最后一个span子元素 */
        .demo > span:not(:last-child){
           font-size: 30px;
        }

example： 
.first span:nth-child(2n+1){
    /* 此时选择的是first类中的所有后代中 排在奇数位的span元素 
    	即如果此时first类中第三个直接子元素不是span类型的元素则不被选中
    也就是所除了span元素外的后代元素也参与排列
    */
}

.first span:nth-of-type(2n+1){
    /* 此时选中的是first类中的所有后代 排在奇数位的span元素
    	但是此时只有后代元素是span类型才参与排列
    */
}

div:only-child{
    /* 此时选择只有一个孩子元素的div中的孩子 */
}

main:only-of-type{
    /* 此时选中mian标签中 所有没有同类型兄弟的后代元素 */
}

6.元素的伪类

元素的伪类： 用于指定特殊元素 
			如 a标签的 link visited（这两个只有a标签能用） hover active
  		/* 
            :link 用来表示没访问过的链接（正常的链接）
         */
        a:link{
            color: red; 
        }
        /* 
            :visited 用来表示访问过的链接
            由于隐私的原因，所以visited这个伪类只能修改链接的颜色
        */
        a:visited{
            color: orange; 
            /* font-size: 50px;   */
        }

         button:hover{
             /* 点击按钮时按钮颜色变红 */
             color: red;
         }

         button:active{
             /* 点击按钮是按钮变大 */
             width: 100px;
         }

7.伪元素选择器

伪元素选择器：	表示页面中不真实存在的元素（特殊位置）
				伪元素 使用 :: 开头
	            ::first-letter 表示第一个字母
                ::first-line 表示第一行
                ::selection 表示选中的内容
                ::before 元素的开始 
                ::after 元素的最后
                    - before 和 after 必须结合content属性来使用
/* 使用伪元素选择器 选择 id 为 demo 的p标签内文本段的第一个字母 */
        #demo::first-letter{
            /* 选中文本段第一个字母 */
            font-size: 30px;
            color: blue;
        }

        /* 使用伪元素选择器 选择 id 为 demo 的p标签内文本段的开始 */
        #demo::before{
            /* 选中文本段的起始 */
            content: '##';
        }
         /* 使用伪元素选择器 选择 id 为 demo 的p标签内文本段的末尾 */
         #demo::after{
            /* 选中文本段的末尾 */
            content: '##';
        }

	使用 ::before 和 ::after 时 加入的文本content也算第一个字母！！！
