003-列表-超链接-图片-音视频

2022-02-25 HTML-CSS学习 1.1k 字
WEB前端

1.列表

​ html中分为三种列表：有序，无序，定义列表

<!-- 无序列表 unordered list-->
<ul>
    <li>序号1</li>
    <li>序号2</li>
    <li>序号3</li>
</ul>


<!-- 有序列表 ordered list-->
<ol>
    <li>无序1</li>
    <li>无序1</li>
    <li>无序1</li>
</ol>

<!-- 定义列表 description list
	使用dl创建定义列表 使用dt表示定义名 dd对定义进行说明
-->
<dl>
    <dt>定义名</dt>
    <dd>解释说明</dd>
</dl>

日常中多使用无序列表 ul标签 在li标签中可嵌套使用 ul标签 实现列表嵌套

2.超链接

​ 使用超链接标签可是将页面跳转到指定页面 或者是当前页面的指定位置

<!-- 
	使用a标签 
		使用属性href 指定跳转的目标 目标可以是指定网站 也可以是服务器内部页面（使用相对路径）
		超链接是一个行内元素 在a标签内可以嵌套除了自身外的所有标签
-->
<a href="http://www.baidu.com">这是一个超链接</a>

<!--
	href 的target属性：
		_self: 超链接的默认属性 不写也可 效果为在当前网页打开链接
		_blank: 保留当前页面 在新的页面打开链接
	
	href的几个常用值：
		1.javascript:;  -> 此时点击链接什么都不会发生
		2.#  -> 此时点击链接会自动回到网页顶部
		3.标签id -> 此时点击链接将跳转到指定id标签的位置 使用标签id时 尽量不要重复id
-->
<a href="www.baidu.com" target="_blank">点我在新页面打开链接</a>

3.图片标签

​ 使用 img 标签向当前页面引入图片资源（外部图片）img 标签属于替换元素（介于块元素和内联元素之间）

​ img 标签 默认不会独占一行

<!-- 
	img是自结束标签
	img 的属性：
		1.src：用于引入外部图片路径
		2.alt: 图片的描述 当浏览器加载不到图片时显示alt 搜索引擎使用alt来识别图片
		3.width: 图片的宽度（px）
		  height: 图片的高度 当图片的宽度或高度变化时 图片将按等比例缩放图片宽高
-->
<img src="在这里引入图片路径" width="200px/可不带单位" alt="这是图片的描述">

src的资源路径可以是base64编码字符串

4.内联框架

​ 使用 iframe 标签 用于在该页面引入其他页面 iframe 元素属于行内元素

<!--
	iframe 属性：
		1.src：用于引入网页路径
		2.frameborder：指定内联框架的边框
-->
<iframe src="https://www.bilibili.com" width="1000" height="600" frameborder="1">不支持iframe时将显示这行字！</iframe>

5.音视频

​ 使用 audio 标签在页面引用外部音频文件

<!--
        属性：
            controls 是否允许用户控制播放 默认情况下不允许用户自己控制播放停止
            autoplay 音频文件是否自动播放
                - 如果设置了autoplay 则音乐在打开页面时会自动播放
                    但是目前来讲大部分浏览器都不会自动对音乐进行播放 
            loop 音乐是否循环播放
		
		还可通过在标签内使用source标签引入音频资源 这样的好处是可以是浏览器兼容解析合适的文件
			还可以在这里写入浏览器不支持audio标签的提示
		embed标签为IE8以下的浏览器所解析使用

		source 和 embed 标签均为自结束标签！ 
		embed 标签内 的type 为map 匹配 文件类型（audio/video）/文件后缀（mp3/mp4）
-->
<audio src="./source/这里是路径.mp3" controls autoplay loop></audio>
 <audio controls>
        <source src="./source/audio.mp3">
        <source src="./source/audio.ogg">
        <embed src="./source/audio.mp3" type="audio/mp3" width="300" height="100">
     	<!--以上资源默认从上往下解析 对不起，您的浏览器不支持播放音频！请升级浏览器！ -->
 </audio>

<!-- 
        使用video标签来向网页中引入一个视频
            - 使用方式和audio基本上是一样的
     -->
 <video controls>
        <source src="./source/flower.webm">
        <source src="./source/flower.mp4">
        <embed src="./source/flower.mp4" type="video/mp4">
    </video>
