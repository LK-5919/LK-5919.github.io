005-样式继承-选择器权重-单位-颜色

1.样式的继承

​ 为一个元素设置的样式会自动应用到他的所有后代子元素身上

​ 注意：并不是所有样式都会继承，如 背景，布局相关样式都不会继承

.box{
           /* box中的颜色样式会自动继承到box的所有子类身上 */
           color: blue;

           width: 200px;
           height: 200px;

           /* 背景颜色不会继承到后代中 但是后代子元素的背景默认为透明 所以看上去同父元素背景颜色一致*/
           background-color: gray;
       }

       /* 选择box中的所有span字元素 */
       div > span, div > p >span{
           /* background-color: red; */
           /* 调整透明度 */
           opacity: .5; 
       }

2.选择器的权重

内联样式        1,0,0,0
id选择器        0,1,0,0
类和伪类选择器   0,0,1,0
元素选择器       0,0,0,1
通配选择器       0,0,0,0
继承的样式       没有优先级

比较优先级时，需要将所有的选择器的优先级进行相加计算，最后优先级越高，则越优先显示（分组选择器是单独计算的）,
      选择器的累加不会超过其最大的数量级，类选择器在高也不会超过id选择器
                如果优先级计算后相同，此时则优先使用靠下的样式

可以在某一个样式的后边添加 !important ，则此时该样式会获取到最高的优先级，甚至超过内联样式，
       注意：在开发中这个玩意一定要慎用！

 /* 选择box中的第一个p元素设置样式 */
         .box :first-child{
             /* 这里的第一个p元素不生效 但是第二个p元素的span元素样式生效 因为span是p标签的中的第一个子元素 */
             color: red;
         }
         /* 选中id为 demo的span标签 */
         #demo{
             /* id选择器优先级高于类选择器和伪类选择器 */
             color: aqua;
         }

<div class="box" >
        我是一个类名叫box的div
        <!-- p元素中的内联样式优先级最高 -->
        <p style="color: skyblue;">
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. Deserunt repellat vero, eveniet dolore ducimus temporibus iure inventore. Nihil ea blanditiis delectus repellat fuga corrupti? Minus voluptate commodi asperiores aspernatur quam!
        </p>

        <p>
            <span id="demo">我是p元素中的span</span>
            <br><br>
            <span>我是p元素中的第二个span元素</span>
        </p>
    </div>

3.像素单位

（1）像素 -- px
（2）百分比 -- %
（3）em -- 相对于以元素的字体大小来计算 默认字体大小是16px
			1em = 1font-size  em根据所在元素字体大小计算而改变
（4）rem -- 根据根元素（root -- html）的字体大小来改变

.demo{
            /* 设计一个box */
            width: 400px;
            height: 400px;
            background-color: green;
        }
        .test{
            /* 这是在demo里的的一个div 大小根据父容器大小改变 */
            width:50%;
            height: 50%;
            background-color: bisque;
            /* 改变元素字体大小 */
            font-size: 12px;
        }
        .t1{
            /* em根据所在元素字体大小改变 */
            font-size: 30px;
            width: 5em;
            height: 5em;
            background-color: blue;
        }
        .t2{
            /* rem根据body根元素的字体大小改变 */
            width: 5rem;
            height: 5rem;
            background-color: red;
        }
        html{
            font-size: 25px;
        }
  <div class="demo">
        <div class="test">
            <div class="t1">
                <div class="t2"></div>
            </div>
        </div>
    </div>

4.颜色

颜色单位：
                   在CSS中可以直接使用颜色名来设置各种颜色
                       比如：red、orange、yellow、blue、green ... ...
                       但是在css中直接使用颜色名是非常的不方便

                   RGB值：
                       - RGB通过三种颜色的不同浓度来调配出不同的颜色
                       - R red，G green ，B blue
                       - 每一种颜色的范围在 0 - 255 (0% - 100%) 之间
                       - 语法：RGB(红色,绿色,蓝色)

                   RGBA:
                       - 就是在rgb的基础上增加了一个a表示不透明度
                       - 需要四个值，前三个和rgb一样，第四个表示不透明度
                           1表示完全不透明   0表示完全透明  .5半透明

                   十六进制的RGB值：
                       - 语法：#红色绿色蓝色
                       - 颜色浓度通过 00-ff
                       - 如果颜色两位两位重复可以进行简写  
                           #aabbcc --> #abc
                   
                   HSL值 HSLA值
                       H 色相(0 - 360)
                       S 饱和度，颜色的浓度 0% - 100%
                       L 亮度，颜色的亮度 0% - 100%
				  A 透明度 0% - 100%
div{
           margin-right: 20px;
       }
       .demo1{
           float: left;
           width: 100px;
           height: 100px;
           background-color: red;
       }
       .demo2{
           float: left;
           width: 100px;
           height: 100px;
           /* 三个值依次表示 红 黄 蓝 */
           background-color: rgb(255, 0, 0);
       }
       .demo3{
           float: left;
           width: 100px;
           height: 100px;
           /* 四个值依次表示 红 黄 蓝 透明度（0~1） */
           background-color: rgba(255, 0, 0, 1);
       }
       .demo4{
           float: left;
           width: 100px;
           height: 100px;
           /* 使用16进制表示 红黄蓝依次使用两个16进制数表示 当每个颜色的两位数字相同时可以采用只写一位的简写 */
           background-color: #f00;
       }
       .demo5{
           float: left;
           width: 100px;
           height: 100px;
           /* 使用 hsl 表示颜色 时三个值依次表示 色相 饱和度 亮度 hsl中的色相取值在0~360 可看成一个圆盘 0和360都是红色
               饱和度 和 亮度 的取值在0%~100%
           */
           background-color: hsl(360, 100%, 50%);
       }
       .demo6{
           float: left;
           width: 100px;
           height: 100px;
           /* 使用 hsla 表示颜色 时四个值依次表示 色相 饱和度 亮度 透明度 hsl中的色相取值在0~360 可看成一个圆盘 0和360都是红色
               饱和度 和 亮度 透明度 的取值在0%~100%
               透明度 0% 表示不透明 此时元素不显示
           */
           background-color: hsl(360, 100%, 50%, 100%);
