006-文档流-盒模型

1.文档流（normal flow）

​ 在文档流中，内联元素默认从左到右流，遇到阻碍或者宽度不够的自动换行，继续按照从左到右的方式布局；块级元素独占一行，按照从上到下的方式布局。

​ 文档流是网页的基础 所创建的元素默认在文档流中排列

元素在文档流中有什么特点：
        - 块元素
            - 块元素会在页面中独占一行(自上向下垂直排列)
            - 默认宽度是父元素的全部（会把父元素撑满）
            - 默认高度是被内容撑开（子元素）

        - 行内元素
            - 行内元素不会独占页面的一行，只占自身的大小
            - 行内元素在页面中左向右水平排列，如果一行之中不能容纳下所有的行内元素
                则元素会换到第二行继续自左向右排列（书写习惯一致）
            - 行内元素的默认宽度和高度都是被内容撑开

2.盒模型（Box Model）

盒模型 ： 可以将网页中的任何元素都看成一个矩形
		每一个盒子都由以下几个部分组成：
		1.内容区（content）
		2.内边距（padding）
		3.边框（border）
		4.外边距（margin）

	内容区 内边距 边框 都会影响可见盒模型的大小！！！

1.内容区（content）

内容区的大小由 width 和 height 两个属性设置
/* 设置 demo 盒子的大小 */
        .demo{
            width: 200px;
            height: 200px;
            background-color: #bfa;
        }

2.边框（border）

要想在网页的盒子显示边框 必须要设置三个属性 border-width border-color border-style
		border-width 和 border-color 有默认值 可省略不写（border-width默认3px border-color默认使用盒子的color）
	（1）border-width：可用来指定盒子四个方向的边框宽度 有默认值 3px
				值的情况：四个值 ----上 右 下 左 （顺时针）
						三个值 ---- 上 左右 下
						两个值 ---- 上下 左右
						一个值 ---- 上下左右
		border-xxx-width： 可单独指定任意一边的宽度 xxx---top right  bottom  left
	（2）border-color: 可用来指定盒子边框的颜色 有默认值 使用color值
				使用规则与border-width 基本相同
				也有border-xxx-color可使用
	（3）border-style ： 用来指定边框的样式 没有默认值 none
				solid 表示实线
                  dotted 点状虚线
                  dashed 虚线
                  double 双线
			
border简写属性，通过该属性可以同时设置边框所有的相关样式，并且没有顺序要求
		border: red 10px solid (三者顺序任意 需用空格间隔开)
 除了border以外还有四个 border-xxx
                       border-top
                       border-right
                       border-bottom
                       border-left   用来单独设置每一条边框的具体样式
			也有 border-xxx-style 可使用

 		/* 设置demo1 盒子的边框样式 */
        .demo1{
            width: 200px;
            height: 200px;
            background-color: yellowgreen;

            /* 设置边框样式 */
            border-width: 10px;
            border-color: red;
            border-style: solid;

            /* 此时的盒子大小为 边框 + 内容 为 220 * 220  */
        }
          /* 设置demo2 盒子的边框样式 */
          .demo2{
            width: 200px;
            height: 200px;
            background-color: yellowgreen;

            /* 设置边框样式 */
           border: 10px red solid;

            /* 此时的盒子大小为 边框 + 内容 为 220 * 220  */
        }
         /* 设置demo3 盒子的边框样式 */
         .demo3{
            width: 200px;
            height: 200px;
            background-color: yellowgreen;

            /* 设置边框样式 */
           border-style: solid;
           
           border-top-style: double;

           /* 此时边框宽度使用默认值 边框颜色使用默认值 black */

            /* 此时的盒子大小为 边框 + 内容 为 220 * 220  */
        }

3.内边距（padding）

​ 内容区和边框之间的是内边距区 一共有四个方向的内边距 padding-top/right/bottom/left

​ 内边距的设置会影响盒子的大小 背景颜色会延伸到内边距区 内边距区不能单独设置背景颜色

padding 内边距的简写属性同 border-width 相同 可同时指定四个方向的内边距大小

 /* 设置demo1 盒子的外边距 */
        .demo1{
            width: 200px;
            height: 200px;
            background-color: #bfa;


            /* 设置内边距 20px 此时盒子的高度为220px 盒子的背景颜色延伸到内边距区*/
            /* padding-top: 20px; */
            /* 内边距区不能单独设置背景颜色 */

            /* 直接设置4个方向的内边距大小 */
            padding: 10px 20px 30px 40px;

            border: red 10px solid;
        }

        .demo2{
            width: 200px;
            height: 200px;
            background-color: black;
            /* demo2盒子在demo1盒子中 大小与demo1的内容区大小一致 此时设置demo1 的内边距 直接影响demo1
                盒子的整体大小
                此时demo1的整体高度为240px
            */



            padding-top:10px;

            /* 设置demo2 盒子的内边距不影响demo1的盒子大小 此时 设置demo2 内边距 padding-top 10px 
                此时demo2 整个盒子的高度变为210px
                因为demo1此时边框距demo2 的边框还差 20px 所以两者边框未重叠
            */

            border: blue 10px solid;

            /* 此时设置；两盒子的边框 demo2 的盒子高度达到230px 此时与demo1 的边框重合 
                demo2 此时覆盖了demo1 的padding-bottom 30px    
            */
        }

	<div class="demo1">
        <div class="demo2"></div>
    </div>


盒子的可见框的大小，由内容区 内边距 和 边框共同决定，所以在计算盒子大小时，需要将这三个区域加到一起计算

4.外边距（margin）

​ 外边距不会影响盒子可见框的大小 只影响盒子排放的位置 一共有四个方向的外边距

                 margin-top
                            - 上外边距，设置一个正值，元素会向下移动
                 margin-right
                            - 默认情况下设置margin-right不会产生任何效果
                 margin-bottom
                            - 下外边距，设置一个正值，其下边的元素会向下移动
                 margin-left
                            - 左外边距，设置一个正值，元素会向右移动
                - margin也可以设置负值，如果是负值则元素会向相反的方向移动

                - 元素在页面中是按照自左向右的顺序排列的，
                    所以默认情况下如果我们设置的左和上外边距则会移动元素自身
                    而设置下和右外边距会移动其他元素

                - margin的简写属性
                    margin 可以同时设置四个方向的外边距 ，用法和padding一样

                - margin会影响到盒子实际占用空间

		/* 设置demo1盒子的外边距样式 */
        .demo1{
            width: 200px;
            height: 200px;
            background-color: #bfa;

            /* 设置外边框 */
            border: 10px red solid;

            /* 设置外边距大小 */
            margin: 30px 20px 50px 100px;

            /* 从效果可知此时盒子在设置margin-top 和 margin-left 时才会移动自身的位置 */
        }

        .demo2{
            width: 200px;
            height: 200px;
            background-color: gainsboro;

            /* 调整demo1 的 margin-bottom 时 demo2 的位置会下移 */
             margin: 10px;

            /* 此处其实有垂直方向上的 两个盒子 的margin-bottom 和 margin-top 重叠 */
        }

<div class="demo1"></div>
<div class="demo2"></div>

5.盒模型的水平布局

块元素独占一行 元素在其父元素中的水平方向的位置由以下几个属性共同决定
					  margin-left
                        border-left
                        padding-left
                        width
                        padding-right
                        border-right
                        margin-right
	刚好是盒子从外到内在到内 从左到右的水平宽度之和
一个元素在其父元素的水平布局中必须满足以下等式：
margin-left+border-left+padding-left+width+padding-right+border-right+margin-right = 其父元素内容区的宽度 （必须满足）

若以上等式右边之和小于父元素宽度 称为 过度约束 此时等式会自动调整
		调整原则：1.这七个值中没有为 auto 的值的时候 浏览器自动调整margin-right值使等式成立
				2.如果 width margin-left margin-right 三个值中有为 auto 值的 浏览器会自动优先调整auto使等式成立
					如果 width 和 margin-left/right 均设置为auto 则宽度会默认调整到最大 margin为0
					如果只有连个外边距为auto width固定值 此时会自动调整到margin-left=margin-right
							实现元素在父元素中居中

 	.demo{
            width: 1000px;
            height: 400px;
            background-color: #bfa;
        }
        .demo2{
            width: 300px;
            height: 200px;
            background-color: yellowgreen;
            /* 此时demo2的宽度300px 就算小于其父元素的1000px
                两个demo2共同排放在其父元素demo中 也不可能排放在同一排

                此时 过度约束 成立 没有设置auto值 浏览器调整 demo2的margin-right 为 700px 使等式成立 

                此时demo2的宽度总和占满以父元素 所以 demo3 自动被挤出父元素换行
            */
        }
        .demo3{
            width: 300px;
            height: 200px;
            background-color: yellow;

        }
        .demo4{
            width: 1000px;
            height: 400px;
            background-color: #bfa;
        }
        .demo5{
            /* width 和 margin 均为 auto 此时width为最大 margin为0 */
            width: auto;
            margin: auto;
            height: 200px;
            background-color: yellowgreen;
        }
        .demo6{
            /* width 为 400px ， margin 均为 auto 此时；水平两边margin值相等 实现元素在父元素中居中 */
            width: 400px;
            margin: auto;
            height: 200px;
            background-color: yellowgreen;
        }

        <div class="demo2"></div>
        <div class="demo3"></div>
    </div>

        <div class="demo5"></div>

    </div>

6.盒模型的垂直布局

默认情况下父元素的高度被内容撑开
 .demo{
            width: 500px;
            background-color: #bfa;
            /* 父元素不设置高度 */
        }
        .test{
            width: 200px;
            height: 200px;
            background-color: green;
            /* 子元素高度设置为200px 此时父元素的高度被撑开 为200px */
        }
<div class="demo">
        <div class="test"></div>
    </div>

子元素在父元素的内容区中排列 如果子元素大小超过父元素 称子元素从父元素中 溢出（overflow）
	使用 overflow 属性处理溢出的子元素

				可选值：
                        visible，默认值 子元素会从父元素中溢出，在父元素外部的位置显示
                        hidden 溢出内容将会被裁剪不会显示
                        scroll 生成两个滚动条，通过滚动条来查看完整的内容
                        auto 根据需要生成滚动条
 .demo1{
            width: 500px;
            height: 200px;
            background-color: #bfa;
           

            /* overflow属性应写在父元素中 */
             /* overflow: visible;  */
            /* overflow默认值为 visible 即显示溢出不作处理 */
            /* overflow: hidden; */
            /* overflow: scroll; */
            /* overflow: auto; */
            
        }
        .test1{
            width: 200px;
            height: 400px;
            background-color: yellowgreen;
            /* 此时子元素高度超过父元素 从中溢出 */
        }
<div class="demo1">

    </div>

                overflow-y:调整垂直溢出

7.外边距的折叠

盒子的相邻垂直方向外边距（margin）会发生重叠现象
	1.兄弟元素 间的相邻垂直外边距取两者之间最大值（当两者都为正值时）
		相邻两外边距一正一负时取两者之和
		若两外边距都是负值时取绝对值大的

		.demo1{
            width: 200px;
            height: 200px;
            background-color: #bfa;
            margin-bottom: 50px;

            /* 将demo1 的margin-bottom 设置为 负值 */
            margin-bottom: -30px;


            /* 此时两盒子的垂直外边距均为负值 取demo1 的外边距 -30 px （绝对值较大） */
        }
        .demo2{
            width: 200px;
            height: 200px;
            background-color: yellowgreen;

            /* 此时demo1的margin-bottom值为50px
                设置demo2 的margin-top 为50px
            */
            margin-top: 50px;
            /* 可知发生外边距折叠 此时盒子布局和只设置demo1 margin-bottom 50px 效果一致 */

            margin-top: 100px;
            /* 若将demo2的 margin-top 设置为100 px 两盒子间的间隔 取demo2
                    外边距（100px较大）
            */

            margin-top: -20px;
            /* 若将demo2 的margin-top 值设置为 -20 px 此时两盒子的间隔为 50px-20px=30px 取两者之和 */
        }

<div class="demo1"></div>
<div class="demo2"></div>

实际开发中 兄弟间的边距折叠现象是有利的 不用去处理

	2.父子元素相邻外边距
			子元素的外边距会传递给父元素
			父子外边距折叠会影响页面布局 需要处理

        /* 父子间元素 */

        .demo3{
            width: 500px;
            height: 500px;
            background-color: #bfa;
            /* 设置父元素的边框 */
            border: 2px red solid;
            /* 给父元素设置边框后 就将父子的边框分离开来 此时 子元素的外边距不会传递给父元素 */
        }
        .demo4{
             border: 2px red solid;
            /* 只设置子元素的边框无法分离父子元素的边框 需要在父元素处设置边框 */
            width: 250px;
            height: 250px;
            background-color: yellowgreen;

            /* 设置子元素的外边距 */
            margin-top: 50px;
            /* 可知此时子元素的外边距与父元素的外边距重叠 设置子元素外边距时父元素的外边距也一起变化 */           
        }
 	<div class="demo3">
        <div class="demo4"></div>
    </div>

8.行内元素的盒模型

行内元素不支持设置宽度和高度
	行内元素可以设置padding border margin 都是垂直方向均无效 不影响布局
		.demo1{
            width: 200px;
            height: 200px;
            /* 设置高度和宽度均无效 */
            background-color: #bfa;
            margin-top: 1000px;
            /* 设置垂直外边距无效 */
            margin-left: 100px;
            margin-right: 100px;
            /* 设置水平外边距有效 */
        }
        .demo2{
            width: 200px;
            height: 200px;
            /* 设置高度和宽度均无效 */
            background-color: yellowgreen;
            margin-top: 1000px;
            /* 设置垂直外边距无效 */
            margin-left: 100px;
            margin-right: 100px;
            /* 设置水平外边距有效 相邻水平间距取两者之和*/
        }
 <span class="demo1">我是类名为demo1 的span</span>
 <span class="demo2">我是类名为demo2 的span</span>

行内元素的两个属性：
	（1）display 用来设置元素显示的类型
					  inline 将元素设置为行内元素
                        block 将元素设置为块元素
                        inline-block 将元素设置为行内块元素 
                                行内块，既可以设置宽度和高度又不会独占一行
                        table 将元素设置为一个表格
                        none 元素不在页面中显示
	（2）visibility 用来设置元素的显示状态
   					  visible 默认值，元素在页面中正常显示
                        hidden 元素在页面中隐藏 不显示，但是依然占据页面的位置	

  .box1{
            width: 200px;
            height: 200px;
            background-color: #bfa;
            display: inline;
            /* div变成行内元素后 宽度和高度设置将失效 */
        }
  <div class="box1">我是一个变成行内元素的div</div>
  <div class="box1">我是一个变成行内元素的div</div>

.test1{
            background-color: #bfa;
            display: block;
            /* 将span的display属性设置为 block span将变为块元素 独占一行 */
        }
 .test2{
            background-color: palegoldenrod;
            display: none;
            /* display 属性设置为 none 元素将隐藏 */
        }
 .test3{
            background-color: pink;
            visibility: hidden;
            /* visibility属性设置为 hidden 将元素隐藏 但还是在网页占据位置
                display ： none 元素不再网页中占据位置空间
            */
        }

9.盒子的尺寸

默认情况下 盒子可见框有 content padding border 三者决定
	box-sizing 用来设置盒子尺寸的计算方式（即设置 width 和 height 的作用）
		可选值：content-box 为默认值 此时设置的 width 和height 均为content大小
				border-box 此时width 和 height 设置整个盒子的可见框大小
		.demo{
            width: 200px;
            height: 200px;
            background-color: #bfa;
            border: 10px red solid;
            box-sizing: content-box;
            /* 默认值 此时width 和 height 为content 大小 */
        }
        .demo1{
            width: 200px;
            height: 200px;
            background-color: #bfa;
            border: 10px red solid;
            box-sizing: border-box;
            /* 此时width 和 height 决定整个盒子可见框大小 */
        }

10.轮廓和圆角

（1）轮廓：outline 用来设置元素的轮廓线 用法和border一致
			轮廓和边框不同的是轮廓不会影响可见框的大小
.demo{
            width: 200px;
            height: 200px;
            background-color: #bfa;
            /* border: 10px red solid; */
            outline: 10px red solid;
            /* outline 属性不影响页面布局 
                outline会遮住下面的行内元素 但是border不会
            */ 
        }
<div class="demo"></div>
    <span>hello</span>

（2）阴影：box-shadow 用来设置元素的阴影效果 阴影不会影响页面布局
			   第一个值 水平偏移量 设置阴影的水平位置 正值向右移动 负值向左移动
                第二个值 垂直偏移量 设置阴影的水平位置 正值向下移动 负值向上移动
                第三个值 阴影的模糊半径
                第四个值 阴影的颜色
 	.demo2{
            width: 200px;
            height: 200px;
            background-color: #bfa;
           box-shadow: 20px 5px 30px rgba(0, 0, 0, .4);
        }
（3）圆角： border-radius 用来设置圆角 圆角设置的圆的半径大小
			/* border-top-left-radius:  */
            /* border-top-right-radius */
            /* border-bottom-left-radius:  */
            /* border-bottom-right-radius:  */
	 border-radius 可以分别指定四个角的圆角
                    四个值 左上 右上 右下 左下
                    三个值 左上 右上/左下 右下 
                    两个个值 左上/右下 右上/左下  
	.demo3{
            width: 200px;
            height: 200px;
            background-color: #bfa;
            /* border-radius: 30px; */
            /* 50% 为圆形 */
            /* border-radius: 50%; */

            border-radius: 10px / 55px;
            /* 设置椭圆角 第一个值为 垂直半轴 第二个值为 水平半轴*/
        }

3.练习

line-height 值设置为与父元素的height相等时 此时可让子元素中的字体在父元素中垂直居中
text-decoration ：none 为去除超链接下划线样式

padding的设置增加的大小属于盒子 margin的设置增加的大小不属于盒子
display：inline-block 将元素转换成行内块元素
重温伪元素选择器 ::before{}

设计页面时先考虑整体布局 先实现整体布局效果 再进一步修改每处细节样式
